# todolist(미완성)

▶ 개요 및 현황
- 자바스크립트 기초 강좌를 듣고, 만들어본 단순한 투두리스트 입니다.
- (추가예정) 리스트 항목의 상태에 따라 리스트 목록을 다르게 구현하는 부분이 미구현되어 있습니다.


▶ 기능
- 리스트 추가 기능
- 리스트 삭제 기능
- 완료 리스트 체크 기능
